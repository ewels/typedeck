Lines
Each non-empty line is typed in turn. First press types line 1, next press line 2, etc., looping back to the start. Template variables: {date} = YYYY-MM-DD, {time} = HH:MM:SS, {clipboard} = current clipboard text, {counter} = press count. Use {{ and }} for literal braces.
Instant type
Pastes the whole text via the system clipboard (your previous clipboard contents are restored). Good for large texts.
Safety
If unchecked, pressing again will queue another message.
Timing
Natural typing
